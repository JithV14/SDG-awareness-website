Sitemap Page

Technical Description

The sitemap in is designed using an SVG element to visually map out the website's structure. Each page is represented by a clickable rectangle, labeled with text to indicate its purpose. Lines with arrowheads connect the rectangles, showing how different sections of the site are linked. The colors and hover effects make it more interactive, while the clear layout helps users understand the site's navigation flow at a glance. This design provides both a functional and visually intuitive way to explore the website's structure.

Accessibility

The sitemap is designed to be clear and easy to navigate, with labeled sections and interactive elements that help users find their way around the site

Challenges and Lessons Learned

Challenges that I had to face

  • Ensuring Responsive Design - Ensuring the website remained clear and functional across various screen sizes required the strategic use of flexbox and grid for a responsive and readable layout.
  • Improving Accessibility - Adding alt text, high contrast, and focus indicators was crucial to ensuring the webpage was accessible and user-friendly for everyone
  • Link Structuring Challenge - Ensuring all internal links are correctly formatted and functional so users can navigate seamlessly without broken links.
  • Optimizing the Visual Sitemap Layout - It was very challenging for me to arranging the SVG elements in a clear and structured way as it required careful positioning and alignment.

Lessons learned

  • Balancing Design and Functionality - Realized that a visually appealing website must also be easy to navigate and functional for all users.
  • The Importance of Feedback - Reviewing each other's work and making improvements based on suggestions leads to a polished final product.
  • Understanding SVG - Using SVG allows for scalable, resolution-independent graphics, making it ideal for creating interactive and visually clear designs.

Compliance

  • Accessibility Compliance - The page is constructed using semantic HTML, alt text for images, high contrast colors, and organized navigation, which makes it Web Content Accessibility Guidelines (WCAG) compliant.
  • Security and Data Protection - While the web page itself doesn't store sensitive user data, it does have a contact form, which would require the secure handling of personal data under data protection laws like GDPR.
